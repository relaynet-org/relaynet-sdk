# RelayNet SDK

RelayNet SDK provides client libraries and integration tools
for interacting with **RelayNet Core**, the open-core relay protocol
for cross-organization logistics handoff.

This repository contains **SDKs only**.
It does NOT contain relay rules, state machines, or protocol definitions.

---

## Purpose

The purpose of this SDK is to lower the integration barrier for
clients, partners, and third-party systems that interact with RelayNet.

All relay rules, state transitions, and protocol logic
are defined and governed by **RelayNet Core**.
